### Audit Claude Code Sandbox Configuration

Given the reported sandbox vulnerability, check your exposure:

```bash
# Check if you're running Claude Code with network sandbox enabled
claude config list | grep -i sandbox

# Review what credentials are accessible in your Claude Code environment
env | grep -iE "(token|key|secret|password|auth)" | wc -l

# If you have sensitive creds in env, consider using a .env file
# that's explicitly excluded from Claude Code's sandbox scope

chmod 600 ~/.claude-secrets
```
