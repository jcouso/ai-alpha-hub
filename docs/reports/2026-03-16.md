## 🧪 Research Frontier

### Knowledge Distillation Quality Signals (arXiv: 2603.12270)
New paper challenges the assumption that a teacher LLM's output distribution is always a high-quality training signal for knowledge distillation. The research shows that blindly distilling from larger models can introduce systematic biases — the quality of the training signal matters more than the size of the teacher model.

**Alpha angle:** If you're fine-tuning smaller models using outputs from GPT-5 or Claude as training data (synthetic data generation), this paper suggests you need to filter and validate the teacher's outputs, not just generate at scale. Quality > quantity for distillation.

- Source: [arXiv](https://arxiv.org/abs/2603.12270)

### MADQA Benchmark: AI Agents Match Humans but Don't Reason
New research on the MADQA (Multi-Agent Dynamic Question Answering) benchmark reveals that even the best AI agents match human searchers in raw accuracy but use fundamentally different strategies. Where humans reason strategically about where to find information, agents essentially brute-force search until they stumble on answers.

**Alpha angle:** If you're building agent systems, don't mistake accuracy for capability. Agents that appear to work in demos may be using 10x the API calls a well-designed system would need. The competitive edge isn't making agents smarter — it's making them more efficient in how they search and reason.

- Source: [LLM Watch](https://www.llmwatch.com/p/ai-agents-of-the-week-papers-you-f11)

### Agent Operating Systems: dimos and Hindsight Hit GitHub Trending
Two agent infrastructure projects are trending: `dimos` by dimensionalOS (a "Proxy Operating System" for agents using a "Dimensional Framework") and `Hindsight` by vectorize-io (a learning agent memory system). Both were published/trending March 15-16 and represent a new infrastructure layer between LLMs and tools.

**Alpha angle:** Agent memory and state management is the unsolved infrastructure problem. Current coding agents (Claude Code, Cursor) lose context between sessions. Projects like Hindsight (persistent learning memory) and dimos (agent coordination layer) are early attempts to solve this. If you're building multi-agent systems, watch these repos for patterns — the "agent OS" category is forming right now.


### 2) Request Copilot Code Review from CLI
GitHub shipped CLI-based Copilot code review. You can now request AI review without leaving your terminal:

```bash
# Request Copilot review on a PR
gh pr review --request-copilot

# Or create a PR and immediately request Copilot review
gh pr create --fill && gh pr review --request-copilot
```

Combine with Claude Code Review (launched March 9) for dual-LLM code review — catch different classes of bugs with different models.
