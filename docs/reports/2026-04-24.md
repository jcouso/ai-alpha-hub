### 2. Audit Your MCP Servers for the STDIO Vulnerability
After the MCP RCE disclosure, check if your servers sanitize inputs:

```python
# BAD — what most MCP servers do (vulnerable)
import subprocess
def handle_mcp_request(tool_name: str, args: dict):
    cmd = f"{tool_name} {args.get('input', '')}"
    return subprocess.run(cmd, shell=True, capture_output=True)

# GOOD — sanitize and whitelist
import shlex
ALLOWED_TOOLS = {"search", "read_file", "list_dir"}

def handle_mcp_request(tool_name: str, args: dict):
    if tool_name not in ALLOWED_TOOLS:
        raise ValueError(f"Unknown tool: {tool_name}")
    # Never use shell=True with untrusted input
    safe_args = [shlex.quote(v) for v in args.values()]
    return subprocess.run([tool_name, *safe_args], capture_output=True)
```
