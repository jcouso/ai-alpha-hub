### 3. Qwen3.6-27B — Dense 27B Model Beats 397B MoE on Coding
Alibaba's Qwen team released Qwen3.6-27B, a dense (non-MoE) open-weight model with 27 billion parameters. If you're running coding assistants on your own hardware or want to avoid cloud API costs, Qwen3.6-27B is immediately relevant.
