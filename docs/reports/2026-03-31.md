## 🧑‍💻 Coding Tips

### 1. Inspect your npm packages for source map leaks
The Claude Code leak happened because `.map` files were bundled into the npm package. Check if YOUR published packages have the same problem:

```bash
# Check if your npm package includes source maps
npm pack --dry-run 2>&1 | grep -E '\.map$'

# If you see .map files, add to .npmignore:
echo "*.map" >> .npmignore
echo "**/*.js.map" >> .npmignore

# Or set in tsconfig.json:
# "sourceMap": false  (for production builds)
# "declarationMap": false

# Verify your published package is clean:
npm pack && tar -tzf *.tgz | grep -E '\.map$'
```

### 2. Try Mistral Voxtral TTS (open-weight, self-hostable)
Mistral's new TTS is open-weight and runs on consumer hardware. Quick test:

```python
# pip install mistral-tts  (or via HuggingFace)
from mistralai import Mistral

client = Mistral(api_key="your-key")

# Basic TTS
response = client.audio.speech.create(
    model="voxtral-tts-v1",
    input="Ship it before you're ready.",
    voice="alloy",
    response_format="mp3"
)

# Save the audio
with open("output.mp3", "wb") as f:
    f.write(response.content)

# Zero-shot voice cloning (pass a reference clip)
response = client.audio.speech.create(
    model="voxtral-tts-v1",
    input="This is my cloned voice.",
    voice_reference="path/to/reference.wav"
)
```
