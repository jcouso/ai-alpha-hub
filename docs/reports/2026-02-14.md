## üßë‚Äçüíª Coding Tips

### 1. Xcode MCP Bridge for External Agents
Connect Claude Code or Codex to Xcode's build system:
```bash
# Claude Code
claude mcp add --transport stdio xcode -- xcrun mcpbridge

# Codex
codex mcp add xcode -- xcrun mcpbridge
```
Xcode shows a visual indicator when an external agent is connected.

### 2. Environment-Aware Agent Config
Use `CLAUDE_CONFIG_DIR` to auto-detect Xcode vs pure CLI environments:
```bash
# In your project root, create both:
# CLAUDE-XCODE.md  ‚Äî Xcode-specific agent instructions
# CLAUDE-PURE.md   ‚Äî CLI-specific agent instructions
# Your CLAUDE.md routes to the right one based on env detection
```

### 3. AGENTS.md is Now Universal
As of Feb 2026, `AGENTS.md` (or `CLAUDE.md`) is supported by: **Cursor, Claude Code, GitHub Copilot agent mode, Continue.dev, Aider, OpenHands, Windsurf, and Xcode 26.3**. Write one and every agent reads it.

### 4. Two-Tier Codex Strategy
```
Fast iteration / completions ‚Üí GPT-5.3-Codex-Spark (128k ctx, low latency)
Complex multi-file agent tasks ‚Üí GPT-5.3-Codex (full reasoning)
Deep architectural decisions ‚Üí Claude Opus 4.6
```

### 5. Zero Trust CI/CD ‚Äî Kill Static Secrets
The Feb 2026 Open Source Radar highlights the death of long-lived API keys in CI/CD. Replace `AWS_ACCESS_KEY_ID` in GitHub Secrets with OIDC federation and short-lived tokens. Agent-first architectures need this yesterday.
