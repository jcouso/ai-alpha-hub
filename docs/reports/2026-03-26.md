## 🚀 Top Early Moves

### 1. 🚨 GitHub Copilot Training on Your Code by Default
Starting April 24, GitHub will use interaction data from all Copilot Free, Pro, and Pro+ users to train AI models — opt-out only. Data includes: inputs, outputs, code snippets, associated context, file names, navigation patterns, and feedback. Private repo code is included if Copilot is active in that repo.

**Why it matters:** This reverses GitHub's previous commitment to not train on user data. Private repos aren't truly "private" anymore if Copilot is enabled. Enterprise/Business tiers are exempt. If you're on a personal plan and writing proprietary code, you need to opt out now — or switch to an alternative.

**Opt out:** Settings → Copilot → Uncheck "Allow GitHub to use my data for model improvement"

### 2. Mistral Releases Voxtral TTS — Open-Weight, Frontier Quality
Mistral released Voxtral TTS, the first frontier-quality open-weight text-to-speech model. Based on Ministral 3B, it supports 9 languages (EN, FR, DE, ES, NL, PT, IT, HI, AR). Key specs: 90ms time-to-first-audio, 6x real-time factor, voice cloning from <5 seconds of audio. Runs on smartwatches, phones, laptops. Full weights available for download.

**Why it matters:** This completes Mistral's speech stack (Voxtral Transcribe → LLM → Voxtral TTS) for end-to-end enterprise voice pipelines without external dependencies. If you're building voice features, you can now run frontier TTS locally at a fraction of ElevenLabs' cost. The voice AI API business model just got disrupted.

### 3. ARC-AGI-3 Launches — Interactive Reasoning Benchmark
The ARC Prize Foundation released ARC-AGI-3, an interactive reasoning benchmark with 1,000+ video-game-like environments. Instead of static puzzles, AI agents must explore, discover goals, build world models, and learn continuously. Best AI in preview: 12.58%. Human baseline: 100%.

**Why it matters:** ARC-AGI-2 was about pattern matching. ARC-AGI-3 is about learning. It measures planning horizons, memory compression, and belief updating — the exact capabilities needed for true autonomous agents. The 87% gap between AI and humans is the clearest measure of how far we actually are from AGI.

## 🧪 Research Frontier

### ARC-AGI-3: The "Can Your Agent Actually Learn?" Test
The most important benchmark release of 2026 so far. ARC-AGI-3 tests skill acquisition efficiency over time, not just final answers. Agents face novel environments with no instructions, sparse feedback, and long planning horizons. The 12.58% vs 100% gap is a humbling reality check for the "AGI is near" crowd.

**Alpha angle:** If you're building agents that need to operate in genuinely novel environments (not just follow pre-trained patterns), this benchmark is your scoreboard. The open toolkit lets you test your own agents.
- Docs: https://docs.arcprize.org/

### Google DeepMind: "Agent Skills" to Close the Knowledge Gap
Google published a blog on "agent skills" — structured, reusable knowledge modules that give AI agents up-to-date SDK and API knowledge. Their Gemini API developer skill boosted task performance from 28.2% to 96.6%. This addresses the fundamental problem of LLMs being trained on stale data.

**Alpha angle:** This is directly relevant to anyone building with coding agents. The concept: instead of fine-tuning models for every API update, you package current knowledge as "skills" the agent loads at runtime. This is the same pattern OpenClaw uses — expect it to become an industry standard.
- Google Developers Blog: https://developers.googleblog.com/closing-the-knowledge-gap-with-agent-skills/
