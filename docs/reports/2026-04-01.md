## 🧑‍💻 Coding Tips

### 1. Install the Codex Review Gate in Claude Code
OpenAI's new plugin lets you block Claude Code from finalizing changes until Codex reviews them. Set it up:

```bash
# Install the Codex plugin for Claude Code
claude plugin add codex-plugin-cc

# Run setup (requires ChatGPT subscription or OpenAI API key)
claude codex setup

# Enable Review Gate — blocks merges until Codex approves
claude codex config --review-gate=on

# Trigger adversarial review on current changes
claude codex review --adversarial

# Or delegate a task to Codex in background
claude codex delegate "write comprehensive tests for auth module"
```

### 2. Apply Meta's structured prompting for code review
Meta's technique boosts code review accuracy to 93%. Structure your review prompts like this:

```markdown
## Code Review Template (Meta Structured Prompting)

Review the following diff. Follow these steps IN ORDER:

### Step 1: Evidence Gathering
- List every function called in the diff
- List every variable modified
- Identify all control flow changes

### Step 2: Trace Execution
- For each modified function, trace one happy path and one error path
- Note any missing error handling

### Step 3: Hypothesis
- State what the diff intends to accomplish
- List potential failure modes

### Step 4: Verdict
- APPROVE, REQUEST_CHANGES, or BLOCK
- For each issue: severity (critical/major/minor), file:line, fix suggestion

[DIFF]
{paste diff here}
```
