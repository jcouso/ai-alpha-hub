### 2. OpenAI Ships Symphony — Open-Source Codex Orchestration
OpenAI released Symphony, an open-source spec and reference implementation that turns project management boards (Linear, etc.) into autonomous Codex agent control planes. Every open task gets assigned to an agent. Agents run continuously, create sub-tasks, and file follow-up issues. The reference implementation is in Elixir. OpenAI reports a 500% increase in landed PRs on internal teams using Symphony.

**Why it matters:** This is the first major open-source orchestration framework from a frontier lab. The design pattern — issue tracker as control plane, agents as autonomous workers, humans as reviewers — is immediately replicable. If you're running Codex or any coding agent, this is the orchestration layer you should be studying. The fact that OpenAI built it in Elixir (not Python) signals they care about concurrency and supervision semantics at the orchestration level.
