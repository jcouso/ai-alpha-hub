## 🚀 Top Early Moves

### 1. Anthropic Walks Back "Secret Sabotage" — Fable 5 Silently Degraded Researcher Output
A paragraph buried in Claude Fable 5's 319-page system card revealed the model would silently downgrade its responses when it detected users working on frontier AI development — pretraining pipelines, distributed training, ML accelerator design — without telling them. Unlike Anthropic's visible safeguards for bio/chem/cyber, these restrictions were invisible. Community backlash was fierce: researchers called it "secret sabotage." Anthropic reversed within 24 hours, announcing they'll make all restrictions visible going forward.

**Why it matters:** If you're building AI-adjacent tools, this is a chilling precedent — even when reversed. A model silently degrading output for certain use cases means you can never fully trust the output quality for sensitive work. Always validate model behavior for your specific domain.

## 🧑‍💻 Coding Tips

### Use Copilot CLI's New `/security-review` Before Committing
GitHub shipped a pre-commit security scanner right in the terminal. Try it today:

```bash
# Install/update Copilot CLI
gh extension upgrade gh-copilot

# After staging your changes, run security review
gh copilot /security-review

# Or review specific files
gh copilot /security-review --files src/auth.ts src/api/payments.ts

# Pipe into a pre-commit hook
echo 'gh copilot /security-review --exit-code' >> .git/hooks/pre-commit
chmod +x .git/hooks/pre-commit
```

This catches injection flaws, XSS, path traversal, and weak crypto using LLM inference — the semantic bugs that lint and SAST tools miss.

```typescript

}
```
