### GenCAD: Image → Parametric CAD Program Generation
355 HN points, 91 comments. Research paper from Carnegie Mellon introducing GenCAD — a model that takes an image and generates the full parametric CAD command sequence (not just a mesh or point cloud). ## 🧑‍💻 Coding Tips

### Add Semble to your coding agent RIGHT NOW

This is the highest-ROI tool tip in weeks. 98% fewer tokens means dramatically cheaper and faster agent runs.

```bash
# Add to Claude Code as MCP server (one command)
claude mcp add semble -s user -- uvx --from "semble[mcp]" semble

# Or install standalone and add to AGENTS.md / CLAUDE.md
pip install semble

# Search your codebase in natural language
semble search "authentication flow" ./my-project
semble search "save_pretrained" ./my-project --top-k 10

# Find related code from a known location
semble find-related src/auth.py 42 ./my-project
```

Add this to your `CLAUDE.md` or `AGENTS.md`:
```markdown
## Code Search
Use `semble search` to find code by describing what it does, instead of grep:
semble search "authentication flow" ./my-project
```
