### 1) Pilot Copilot CLI with measurable guardrails

```bash
# example pilot baseline

```

Track: task completion time, accepted suggestion rate, and post-merge bug regressions for 1 sprint.

### 2) Keep PR automation helpful but constrained

```yaml

## AI-assisted summary checklist
- [ ] Title reflects actual change scope
- [ ] Mentions migrations/config changes
- [ ] Lists security-sensitive changes explicitly
```

PR-title generation is useful, but enforce human final pass to avoid vague or misleading merge history.
