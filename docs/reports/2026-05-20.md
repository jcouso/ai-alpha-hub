### Try Gemini 3.5 Flash via API today

GA immediately. 1M context at $1.50/1M input tokens — the cheapest frontier-class model for agent workloads.

```python
import google.generativeai as genai

genai.configure(api_key="YOUR_KEY")

# Gemini 3.5 Flash with thinking
model = genai.GenerativeModel(
    "gemini-3.5-flash",
    generation_config={
        "thinking_config": {"thinking_level": "high"},
        "max_output_tokens": 65536,
    }
)

response = model.generate_content(
    "Analyze this codebase and suggest architecture improvements",
)
print(response.text)
```

### Managed Agents API — spin up an agent with one call

```python
# New Managed Agents API from Google
# Single call → agent with tools + code execution + isolated environment
import google.generativeai as genai

client = genai.Client(api_key="YOUR_KEY")

interaction = client.interactions.create(
    model="gemini-3.5-flash",
    instructions="You are a code review agent. Analyze PRs for bugs and security issues.",
    tools=["code_execution", "web_search"],
)
# Agent persists state across calls — resume with interaction.id
```
