## 🧑‍💻 Coding Tips

### 1. Install MCP Servers from Registry in Copilot CLI
Copilot CLI v1.0.25 now supports guided MCP server installation:

```bash
# Update to latest
npm install -g @github/copilot-cli@latest

# Browse and install MCP servers with guided config
copilot mcp install

# View your loaded environment (new /env command)
# Inside a session, type: /env
```

This replaces manual JSON editing for MCP server configuration. The guided flow handles auth, transport type (stdio/HTTP/SSE), and config validation.
