## üß™ Research Frontier

### Fast KV Compaction via Attention Matching (arxiv 2602.16284)
A new approach to compress KV caches in latent space by constructing compact keys and values that reproduce attention outputs. The method achieves **up to 50x compaction in seconds** on some datasets with little quality loss ‚Äî a massive improvement over previous approaches that required slow end-to-end optimization.

**Why this matters for practitioners:** KV cache size is the real bottleneck for long-context agent workflows. If you're running multi-step agent loops with Claude Code or Codex, KV cache management determines how much context you can carry forward. This paper shows a practical path to dramatically better context compression.

HN with 55 points and growing.

**Source:** [arxiv.org/abs/2602.16284](https://arxiv.org/abs/2602.16284)

### Together AI CDLM: Diffusion-Based LLM Inference at 14x Speed
(See main coverage above.) The key research insight: instead of generating tokens one-by-one (autoregressive), CDLMs iteratively refine masked sequences, generating multiple tokens per step. The block-wise causal masking trick makes this compatible with existing KV caching infrastructure ‚Äî solving the main practical barrier to diffusion LMs.

### Anthropic Agent Autonomy Measurement
(See main coverage above.) The methodological contribution: using Clio (privacy-preserving analysis tool) to study millions of real-world interactions. The finding that agent-initiated pauses outnumber human interrupts on complex tasks is a signal that we're entering an era where the agent is better at knowing when to stop than the human is at knowing when to intervene.
