## 🧑‍💻 Coding Tips

### Check if Claude Code Auto Dream is Active

Claude quietly shipped memory consolidation between sessions. Check if it's running on your setup:

```bash
# Inside a Claude Code session, run:
/memory

# Look for "Auto-dream: on" in the selector
# If active, Claude is automatically:
# - Merging duplicate memory entries
# - Pruning stale notes
# - Reorganizing memory files between sessions

# To see what it's consolidated:
cat ~/.claude/memory/*.md | wc -l
# Compare this over a few days — if the line count stays stable
# or drops while you're actively using Claude Code, dreaming is working
```
