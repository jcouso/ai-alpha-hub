## 🧑‍💻 Coding Tips

### Use GitHub's new Copilot Agent Tasks REST API for CI-triggered coding tasks

```bash
# Start a Copilot cloud agent task via API
curl -X POST \
  -H "Authorization: Bearer $GITHUB_TOKEN" \
  -H "Accept: application/vnd.github+json" \
  https://api.github.com/repos/OWNER/REPO/copilot/agent-tasks \
  -d '{"instructions": "Refactor the auth module to use JWT tokens"}'
```

This unlocks: nightly refactoring runs, automated issue-to-PR pipelines, and programmatic code quality sweeps — all headless.

### Check your MCP server implementations for SQL injection

After The Register's report on three MCP database flaws, audit any MCP servers you're running:

```bash
# Search for potentially unsafe database MCP servers
grep -r "MCP" ~/.config/ ~/code/ --include="*.json" --include="*.yaml" -l

# Key things to check:
# 1. Does your MCP server sanitize inputs before constructing SQL?
# 2. Are database credentials exposed through MCP metadata endpoints?
# 3. Is the MCP server exposed to the internet (vs. localhost only)?
```
