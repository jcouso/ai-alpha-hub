## 🧑‍💻 Coding Tips

### Try Copilot CLI's New `/every` Command for Scheduled Agent Tasks

GitHub just shipped prompt scheduling in Copilot CLI. Set up recurring automated tasks without leaving your terminal:

```bash
# Install/update Copilot CLI
gh extension upgrade gh-copilot

# Schedule a daily dependency audit
gh copilot /every "8am" "Check for security vulnerabilities in package.json and report critical ones"

# Schedule pre-standup code summary
gh copilot /every "9:30am weekdays" "Summarize git log --since=yesterday in 5 bullets"

# Voice input: hold spacebar + speak your prompt
# Audio stays on-device (local speech-to-text)
```

### Model Your AI Tooling Costs (Post-Uber Wake-Up Call)

After Uber's $1,500/mo/employee cap and Copilot's metered billing, here's a quick cost calculator:

```python
# ai_cost_model.py — estimate monthly AI tool spend per engineer
TOOLS = {
    "claude_code": {"avg_tokens_day": 500_000, "input_per_1m": 3, "output_per_1m": 15},
    "copilot_agentic": {"est_monthly": 75},  # new metered avg
    "cursor_pro": {"flat": 20, "premium_requests": 40},  # 500 req + overages
}

engineers = 8  # your team size
days = 22  # working days/month

for tool, pricing in TOOLS.items():
    if "avg_tokens_day" in pricing:
        monthly = days * pricing["avg_tokens_day"] / 1_000_000 * (
            pricing["input_per_1m"] * 0.7 + pricing["output_per_1m"] * 0.3
        )
    elif "est_monthly" in pricing:
        monthly = pricing["est_monthly"]
    else:
        monthly = pricing.get("flat", 0) + pricing.get("premium_requests", 0)
    print(f"{tool}: ${monthly:.0f}/eng/mo → ${monthly * engineers:.0f}/team/mo")

# Output:
# claude_code: $105/eng/mo → $840/team/mo
# copilot_agentic: $75/eng/mo → $600/team/mo
# cursor_pro: $60/eng/mo → $480/team/mo
# TOTAL: ~$240/eng/mo → ~$1,920/team/mo
```

Build this into your client pricing. The era of "AI tools are basically free" ended this week.
