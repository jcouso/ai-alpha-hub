### 3. ARC-AGI-3: GPT-5.5 Scores 0.43%, Opus 4.7 Scores 0.18%
ARC Prize published a deep analysis of 160 reasoning replays from GPT-5.5 and Opus 4.7 attempting ARC-AGI-3's novel environments. Both score under 1%. They identified three common failure modes: (1) models understand local effects but can't build global world models, (2) they mistake novel environments for familiar games from training data, (3) even when they solve a level, they can't transfer learning to the next one. The analysis package is open-sourced.

**Why it matters:** This is the best window we have into *how* frontier models fail at genuine novelty. The open-sourced replays are gold for anyone working on agent architectures — you can watch exactly where the models' reasoning breaks down. Useful for calibrating expectations about what current models can and can't do autonomously.

- Sources: [ARC Prize Blog](https://arcprize.org/blog/arc-agi-3-gpt-5-5-opus-4-7-analysis) · [Scorecards](https://arcprize.org/scorecards/model/openai-gpt-5-5-2026-04-23-high)

## 🧑‍💻 Coding Tips

### 1. Check Your Builds for Leaked Instruction Files
Apple shipped Claude.md files in a production iOS app. Don't be Apple:

```bash
# Find any AI instruction files that shouldn't be in your build
find ./dist ./build ./.next -name "*.md" \
  -exec grep -il "claude\|CLAUDE\|system prompt\|AGENTS\\.md" {} \;

# Add to .gitignore if not already:
echo "CLAUDE.md" >> .gitignore
echo "AGENTS.md" >> .gitignore
echo ".cursor/" >> .gitignore

# For iOS/Android: add to your build exclusion rules
# Xcode: Build Phases → Copy Bundle Resources → remove .md files
# Gradle: android.sourceSets.main.resources.excludes += ["**/*.md"]
```
