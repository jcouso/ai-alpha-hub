## 🧑‍💻 Coding Tips

### Audit your npm lockfile for TanStack supply chain compromise

If you use TanStack Router, Nx Console, or MistralAI npm packages, check for compromised versions immediately:

```bash
# Check for affected TanStack versions
npm ls @tanstack/react-router 2>/dev/null | grep -E "1\.169\.[5-9]"
npm ls @tanstack/router-core 2>/dev/null | grep -E "1\.169\.[5-9]"
npm ls @tanstack/react-start 2>/dev/null | grep -E "1\.167\.(68|71)"

# Check for affected MistralAI packages
npm ls mistralai 2>/dev/null
pip list 2>/dev/null | grep -i mistral
pip list 2>/dev/null | grep -i guardrails-ai

# If any match — force update and rotate all tokens
npm audit fix --force
# Rotate: GitHub tokens, npm tokens, any env vars on affected machines
```

### Try Microsoft's RAMPART for agent security testing

```bash
# Clone and set up RAMPART
git clone https://github.com/microsoft/rampart.git
cd rampart
pip install -e .

# Run a basic agent security scan
# RAMPART tests for prompt injection, tool misuse, privilege escalation
rampart scan --agent-config ./your-agent.yaml --scenarios prompt-injection,tool-misuse

# Clarity — for pre-build agent design review
# Structured questionnaire before you write agent code
git clone https://github.com/microsoft/clarity-agent.git
```
