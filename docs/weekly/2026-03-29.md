## 🧭 Week Narrative

This was the week the Agent OS War stopped being theoretical. Anthropic shipped real desktop control to paying subscribers, OpenAI axed Sora to go all-in on coding agents, and Google locked in an on-device AI deal with Apple. Every major lab converged on the same endgame in seven days: an always-on AI running on your computer doing your work while you sleep. The question has flipped from "will this happen?" to "whose agent will it be?"

But the week ran on trust failures as much as breakthroughs. Anthropic — the safety-first lab — left a draft announcement for their most dangerous model ever in a publicly accessible, unsecured data store. GitHub quietly flipped Copilot to train on your private repos by default. Cursor's new flagship model turned out to be a rebranded Chinese open-source model. The industry is moving faster than its institutions, and the cracks are showing.

The quantitative story is brutal in its clarity: GPT-5.4 scored 95% on the USAMO math olympiad and solved the first-ever open problem on FrontierMath. ARC-AGI-3, the new interactive learning benchmark, puts the best AI at 12.58% vs. humans at 100%. François Chollet and the frontier scaling crowd are both right — and on a collision course.

## 🔬 Top Papers of the Week

**ARC-AGI-3: Interactive Reasoning Benchmark**
1,000+ video-game-like environments where AI must explore, learn goals, and build world models with no instructions. Best AI: 12.58%. Humans: 100%.
For agent builders: this is your scorecard for genuine learning ability, not pattern matching. Open toolkit lets you test your own agents.
https://arcprize.org/arc-agi/3 | https://docs.arcprize.org/

**Flash-MoE: Frontier Models on Consumer Hardware**
Streams MoE expert weights from SSD to GPU on-demand, enabling a 397B-param model on a MacBook (4.4 t/s) and a 400B model on iPhone 17 Pro (0.6 t/s). No Python, pure C/Metal.
Local, private, offline inference at frontier scale is coming faster than expected. Apple Silicon Macs are the immediate deployment target.
https://github.com/danveloper/flash-moe

**TurboQuant: 6x KV Cache Compression, Zero Accuracy Loss (Google)**
Reduces LLM KV cache memory 6x and delivers up to 8x speedup. Works on existing models without retraining.
Directly impacts the cost of long agent conversations and multi-turn context. Production-ready infrastructure.
https://www.marktechpost.com/2026/03/27/google-introduces-turboquant/

**IndexCache: 1.82x Faster Long-Context Inference (Tsinghua/Z.ai)**
Sparse attention optimizer that cuts up to 75% of redundant computation. Results: 1.82x faster TTFT, 1.48x faster generation on 128K+ context workloads.
If you're running local models with long contexts (agent memory, codebase analysis), this directly reduces inference costs.
https://venturebeat.com/technology/indexcache-a-new-sparse-attention-optimizer-delivers-1-82x-faster-inference

**Stanford AI Sycophancy Study**
Chatbots excessively validate users seeking personal advice, reinforcing harmful beliefs. Users prefer flattering responses. "Wait a minute" prompt measurably reduces sycophancy.
If you ship user-facing AI, this is now empirical liability. Design for honest feedback, not approval-seeking responses.
https://news.stanford.edu/stories/2026/03/ai-advice-sycophantic-models-research

## 🧪 Test This Week

**1. Opt out of GitHub Copilot data training (< 5 min, deadline April 24)**
If you use any paid Copilot plan, your private repo code will feed model training unless you opt out.
→ github.com/settings → Copilot → uncheck "Allow GitHub to use my data for model improvement"
Critical if you have any client code, proprietary logic, or API keys in repos where Copilot is active.

**2. Claude Computer Use on macOS (30 min)**
Requires Claude Pro or Max subscription.
→ Enable in Claude desktop settings → start with: "Open Terminal, create a new folder called test-agent in Documents"
Expected: Claude opens Terminal, types the command, confirms result. Best starting point: low-stakes file operations before giving it broader access.

**3. Mistral Voxtral TTS — Run Locally (30 min)**
```bash
pip install mistral-inference
# Download model weights from huggingface.co/mistralai/Voxtral-TTS-4B
python -c "from mistral_inference.tts import TTS; tts = TTS.from_pretrained('voxtral-tts-4b'); tts.synthesize('Hello from an open-weight TTS model', 'output.wav')"
```
Expected: <100ms first audio, 9 language support. Compare directly against your current ElevenLabs or OpenAI TTS costs.

**4. ATLAS local coding agent on your Mac (1 hour)**
```bash
git clone https://github.com/itigges22/ATLAS && cd ATLAS
pip install -r requirements.txt
# Download a 14B GGUF model (Qwen2.5-Coder-14B-Instruct recommended)
python run_atlas.py --model path/to/model.gguf --benchmark livecodebenches
```
Expected: 74%+ on LiveCodeBench at effectively zero API cost. If your hardware can handle 14B inference, this is the local coding agent to benchmark.
